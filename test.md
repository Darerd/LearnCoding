##### git常用指令

``` 
$ mkdir demo		# 创建测试目录
$ cd demo   		# 进入测试目录
$ touch test.md		# 创建test.md文件
$ ls				# 查看目录下的文件
$ git init 			# 初始化
$ git add test.md	# 添加文件到仓库
$ git commit -m "写备注"	# 提交到远程仓库
$ git push origin master	# 推送到Github

#### 把在Github上的更新结果同步到本地文件中 ####
$ git fetch origin
$ git merge origin/master
```
